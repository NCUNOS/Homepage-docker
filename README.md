
Homepage-docker
=====

Dockerize our [Homepage](https://github.com/NCUNOS/Homepage). :v:

Usage
=====

To run it:

```
$ docker run -d -p 80:80 ncunos/Homepage
```

Usage-Compose
=====

docker-compose.yml:

```
luffa:
  image: ncunos/homepage
```
